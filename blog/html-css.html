Margins, Borders, and Padding

Margins, Borders and Padding are properties of elements on a web page that can be adjusted with CSS code. Margins are the space between an element and the surrounding area, padding is the distance between the content and the edge of the containing element, while a border is at the edge of an element. It's okay if that's doesn't make sense much yet. I'll show you how each of each of them work.

Margins

Margins are used to make space between elements on a web page. Using CSS we can set margins using the margin keyword. For example, a margin of 20 pixels could be specified. This means that there is 20 pixels between the edge of the element and the surrounding page or any other adjacent elements.

I have a 20 pixel margin

So do I

This is simply achieved by declaring margin: 20px; in the css rules for the box class. Since each box has a 20 pixel margin in each direction that means there is a 40 pixel gap between them. Margins in specific directions can be specified too using any of the keywords margin-top, margin bottom, margin-left, margin-right. Margins can be given in many different units like cm and inches, and also proportional measurements like the percentage of the page.

Padding

Padding is the internal distance between the content within an element and the edge of an element. For example the words in this box are the content and the padding is the distance between them and the edge of the box

There is a 20px padding between me and my box

This is very similar to the margin except it is for the inside an element as opposed to outside of it. Just like margins, padding can be given in many different units and is simply specified using the keyword padding. Padding can also be given in each direction using the padding-left, padding right, padding-top, padding-bottom keywords

Borders

Borders are at the edge of an element, they are exactly in the middle between the padding and the margin. Borders are used to decorate or emphasize content.

I have a border!

Borders have many properties that can be adjusted with keywords. border-width specifies the thickness of the border. The one above is only three pixels wide but they can be as wide as you want! Another keyword is border-style which specifies what the border will look like. This one is a solid border but it can also be dotted or dashed or any other style of your choice. We can also set the colour of the border using border-color and we can use border-radius to choose how rounded the corners of the border are. The default radius of 0px is just a sharp square corner.
